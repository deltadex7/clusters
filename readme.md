# Clusters

An attempt to build Tetris in C++ using raylib engine.

## Controls

Customizable controls coming soon.

* `F`: clockwise
* `D`: counterclockwise
* `S`: 180 degrees

## Features

[v] 7-piece bag randomizer
[ ] Super/Standard Rotation System (SRS) (50%)
[ ] SRS wallkicks
[ ] 10x20 matrix
[ ] Playable environment
